# countdown_app

## 📝 Description (English)

**Countdown App** is a practical implementation of a training project provided by [Codementor - Countdown Mobile App](https://www.codementor.io/projects/mobile/countdown-mobile-app-bjzn08zcon/discussion).  
This app is designed to help improve Flutter development skills, focusing on UI design, state management, and local data storage using Hive.  
Users can create countdowns for upcoming events with customizable titles, dates, emojis, colors, and background images.
